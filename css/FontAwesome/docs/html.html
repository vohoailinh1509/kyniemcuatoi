Sử dụng Font Awesome

Web font

Link file all.css (hoặc file all.min.css ) vào file html và dùng các thẻ <i> (hoặc thẻ <span> ) với class như sau:

  • fas fa-<tên-của-icon>: Dùng cho các icon được tô kín (solid) hoặc có viền đậm, phần lớn icon trong nhóm này đều free, một số phải trả phí.
  • far fa-<tên-của-icon>: Dùng cho các icon không tô (regular), phần lớn icon trong nhóm này đều phải trả phí, chỉ có một số ít free.
  • fal fa-<tên-của-icon>: Dùng cho các icon không tô và có viền mỏng hơn (light), tất cả icon trong nhóm này đều phải trả phí.
  • fab fa-<tên-của-icon>: Dùng cho các icon là logo của các thương hiệu nổi tiếng (brands), tất cả icon trong nhóm này đều free.

Lưu ý: thư mục /webfonts và /css phải nằm trong cùng đường dẫn, và phải kiểm tra lại đã link đúng đường dẫn tới all.css hay chưa

Cách này chỉ sử dụng 2 thư mục /webfonts và /css nên các thư mục còn lại có thể xóa hoặc di chuyển đi chỗ khác cũng được.

Lưu ý: khi nộp bài vui lòng KHÔNG dùng đường dẫn bắt đầu bằng dấu "/"

Nếu không sử dụng hết các nhóm icon thì có thể link file css của từng nhóm muốn sử dụng vào cùng với file fontawesome.css (hoặc bản min của các file này)

SVG + Javascript

Tương tự như cách trên nhưng dùng file all.js (hoặc file all.min.js)

Cách này chỉ sử dụng thư mục /js nên các thư mục còn lại có thể xóa hoặc di chuyển đi chỗ khác cũng được;

Nếu không sử dụng hết các nhóm icon thì có thể link file js của từng nhóm muốn sử dụng vào cùng với file fontawesome.js (hoặc bản min), lưu ý file fontawesome.js nên để sau các file js của từng nhóm.

  • fa-xs: 0.75em
  • fa-sm: .875em
  • fa-lg: 1.33em (kèm theo vertical-align: -25%)
  • fa-2x đến fa-10x: 2em đến 10em
